## 퀴즈 1 (개념 구분 · 서술형)

다음 설명을 보고 해당되는 개념을 각각 고르시오.
(① 다중공선성 / ② 차원의 저주 / ③ PCA / ④ MDS 중 선택)

1️⃣ 설명변수들 간의 강한 선형 종속성으로 인해
회귀계수의 해석이 불가능해지고, 계수의 분산이 커지는 현상

2️⃣ 차원이 증가할수록 데이터가 공간에서 희박해지며
거리 기반 군집화(K-means, DBSCAN)가 잘 동작하지 않는 현상

3️⃣ 원본 변수들을 선형결합하여 서로 직교하는 새로운 축을 만들고,
이 축으로 데이터를 투영하여 차원을 줄이는 기법

4️⃣ 원본 변수의 의미와는 무관하게,
데이터 간 거리 구조를 최대한 보존하도록 저차원에 배치하는 기법

## 퀴즈 2 (이해·판단형)

다음 문장 중 **옳은 것(O)**과 **틀린 것(X)**을 판단하시오.

1️⃣ PCA를 적용하면 설명변수 간 상관이 제거되므로
다중공선성 문제가 완화된다.

2️⃣ MDS에서 생성된 축은 서로 직교하며,
각 축은 원래 변수의 의미를 유지한다.

3️⃣ SSE와 Silhouette score는 internal evaluation 지표이며,
이를 이용해 K 값을 비교하는 것은 relative evaluation에 해당한다.

4️⃣ 차원의 저주는 주로 회귀계수의 해석 문제를 의미하며,
예측 성능에는 큰 영향을 주지 않는다.

<details> <summary>정답 및 해설</summary>

1️⃣ O
→ PCA는 직교하는 주성분으로 변환하여 공선성을 제거함

2️⃣ X
→ MDS에서 축은 의미 없으며, 회전·반전해도 동일한 해임

3️⃣ O
→ 지표는 internal, 사용 방식이 relative

4️⃣ X
→ 이는 다중공선성의 설명이고,
차원의 저주는 거리·밀도 기반 알고리즘 성능 저하 문제임
